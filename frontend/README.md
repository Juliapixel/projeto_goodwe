# Frontend

Feito com React+Vite, utilizando `react-router` e TailwindCSS, gera
um site SPA estático

## Setup

### Dependências

- node.js
- pnpm

```bash
pnpm i
```

### Desenvolvimento

Desenvolvimento ativo:

```bash
pnpm run dev
```

Antes de commitar:

```bash
pnpm run lint # e consertar os avisos do linter
pnpm run format
pnpm run build # deve finalizar sem erros
```

Preview da build:

```bash
pnpm run build
pnpm run preview
```
